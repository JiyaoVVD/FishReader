# 鱼阅(FishReader)

摸鱼看小说插件。

## Features

工作摸鱼用小说插件，支持在状态栏阅读，隐蔽程度1000%。

[FishReader GitHub Repository](https://github.com/JiyaoVVD/FishReader)


## Extension Settings

* `fishreader.defaultBookPath`: 小说文件路径（目前仅支持UTF-8编码的单个txt文件）
